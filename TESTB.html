General information

ACE is an account creation form (essentially). Managers use this to set up Active Directory accounts for new starters.

Service Desk then verify the AD accounts (but don't check name details etc)

We request ACE access via TABS

We don't offer training (as the training system created by IT no longer works) but we can send a quick reference card
